Условие задачи
Если выписать все натуральные числа меньше 10, кратные 3 или 5, то получим: 3, 5, 6, 9. Сумма этих чисел равна 23.
Напишите функцию, которая возвращает сумму всех чисел, кратных 3 или 5, меньше переданного числа n.


Решение на JavaScript:
function solution(n) {
  let sum = 0;
  for (let i = 3; i < n; i++) {
    if (i % 3 === 0 || i % 5 === 0) {
      sum += i;
    }
  }
  return sum;
}


Условие:
Напишите функцию, которая проверяет правильность расстановки скобок (), [], {}. Строка считается валидной, если все скобки закрыты в правильном порядке.

Решение на JavaScript:
function validBraces(str) {
  const stack = [];
  const pairs = { '(': ')', '[': ']', '{': '}' };

  for (const char of str) {
    if (pairs[char]) {
      stack.push(char); // Открывающую скобку кладем в стек
    } else {
      if (char !== pairs[stack.pop()]) {
        return false; // Закрывающая скобка не соответствует последней открытой
      }
    }
  }

  return stack.length === 0; // Проверяем, что стек пуст
}
